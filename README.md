# AISC Personal Medical Assistant (College Student MVP)

Streamlit MVP for a non-diagnostic personal health assistant focused on common college-student medical issues.

## Safety
- Not a diagnosis.
- Not for emergencies.
- Provides general self-care + OTC info + red-flag escalation guidance ONLY.

## Run locally
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app/main.py
